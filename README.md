# Deep-Pulse-AI-Driven-Tool-Condition-Monitoring
📌 Overview

Deep-Pulse is an intelligent predictive maintenance system that analyzes acoustic and vibration signals from machining tools to detect wear and predict failure in real time.

This project combines Mechanical Engineering principles with advanced Data Science techniques to improve industrial reliability and reduce downtime.

🚀 Features
🎧 Acoustic & vibration signal processing
📊 Spectrogram-based feature extraction
🧠 CNN + LSTM hybrid deep learning model
⏱ Remaining Useful Life (RUL) prediction
📡 Real-time monitoring dashboard (Streamlit)
🔍 Model explainability (Grad-CAM / SHAP)
🏗️ Architecture

Raw Signal → FFT → Spectrogram → CNN → LSTM → Prediction

📂 Dataset
NASA Milling Dataset (Tool wear progression)
Acoustic Emission (AE) & Vibration signals
🧪 Models Used
Convolutional Neural Networks (ResNet-18)
Long Short-Term Memory (LSTM)
Hybrid CNN-LSTM architecture
📈 Results
High accuracy in tool wear classification
Early detection of failure patterns
Reliable RUL prediction trends
